# 快速启动指南

## 🚀 快速开始

### 1. 启动后端服务

首先确保后端Spring Boot服务已启动：

```bash
# 在项目根目录
cd HotelManagementSystem
mvn spring-boot:run
```

后端将运行在 `http://localhost:8080`

### 2. 启动前端服务

```bash
# 进入前端目录
cd app

# 安装依赖（首次运行）
pnpm install

# 启动开发服务器
pnpm dev
```

前端将运行在 `http://localhost:3000`，浏览器会自动打开。

### 3. 登录系统

- **用户名**: 任意输入（例如：admin）
- **密码**: `123456`（固定密码）

## 📋 功能使用指南

### 酒店管理
1. 点击左侧菜单 "酒店管理"
2. 点击 "添加酒店" 按钮
3. 填写酒店信息（名称、地址、电话、星级、描述）
4. 点击 "添加" 保存

**提示**: 建议先添加几家酒店和房间信息，以便后续测试预订功能。

### 房间预订
1. 点击 "房间预订"
2. 选择入住和退房日期
3. 可选填写地点或房型筛选
4. 点击 "搜索可用房间"
5. 在列表中点击 "预订" 按钮
6. 填写客户信息（姓名、电话、邮箱、身份证号）
7. 确认预订

### 订单管理
1. 点击 "订单管理" 查看所有订单
2. 可以根据客户信息、酒店名称或状态搜索
3. 订单操作：
   - ✓ 图标：确认订单
   - ✗ 图标：取消订单
   - 🔓 图标：办理入住
   - 🔒 图标：办理退房
   - 🔔 图标：发送提醒（模拟功能）

### 客户管理
1. 点击 "客户管理"
2. 可以添加、编辑客户信息
3. 点击 👁 图标查看客户详情和历史订单

### 评价管理
1. 点击 "评价管理"
2. 查看所有客户评价
3. 点击 💬 图标回复评价
4. 可以根据评分或回复状态筛选

### 统计报表
1. 点击 "统计报表"
2. 选择开始和结束日期
3. 可选填写酒店ID（留空查询所有）
4. 点击 "查询统计"
5. 查看图表：
   - 预订统计（柱状图）
   - 收入统计（折线图）
   - 入住率统计（折线图）
6. 点击 "导出Excel" 下载报表

## 🎨 界面特色

- **响应式设计**: 适配不同屏幕尺寸
- **现代化UI**: 使用 shadcn/ui 组件库
- **深色主题支持**: 内置深色模式样式
- **图表可视化**: Recharts 数据图表
- **友好的用户体验**: 清晰的操作反馈

## 📝 注意事项

1. **数据库初始化**: 首次运行时数据库为空，需要手动添加数据
2. **后端依赖**: 确保后端服务已启动，否则前端无法获取数据
3. **端口冲突**: 如果3000或8080端口被占用，请修改配置
4. **浏览器兼容**: 建议使用 Chrome、Firefox、Safari 或 Edge 最新版

## 🔧 常见问题

### Q: 无法连接到后端？
A: 检查后端服务是否运行在 8080 端口，可以访问 http://localhost:8080/hotels 测试

### Q: 登录失败？
A: 确保密码输入的是 `123456`，用户名可以任意输入

### Q: 图表不显示？
A: 确保已经查询了统计数据，并且日期范围内有订单数据

### Q: 导出Excel失败？
A: 检查后端日志，确保 Apache POI 依赖正常
